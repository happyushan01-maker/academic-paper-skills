# Academic Paper Workflow

This folder is a working area for planning, researching, drafting, and exporting academic papers.

## Folders

- `papers/` - Store source papers, PDFs, article notes, bibliographies, and other literature materials used during research.
- `data/` - Keep datasets, coding sheets, extracted evidence, survey exports, or any structured research data.
- `scripts/` - Place analysis, validation, conversion, or automation scripts for the paper workflow.
- `outputs/` - Save generated drafts, figures, tables, reports, manuscripts, and submission-ready exports.
- `notes/` - Keep research memos, outline drafts, meeting notes, reviewer feedback, and planning documents.

Each subfolder includes a `.gitkeep` placeholder so the folder is tracked even before project files are added.
